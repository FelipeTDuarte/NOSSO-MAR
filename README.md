# NOSSO-MAR

Neural Operator Scalable System for Ocean Modeling (PIMR-NO + MARL)

See `PHASE_0.md` and `docs/` for architecture and roadmap.

Quickstart

1. Create a Python environment (conda recommended):

```bash
conda env create -f environment.yml
conda activate nosso-mar
```

2. Install editable package (optional):

```bash
pip install -e .
```

3. Run tests:

```bash
python -m pytest -q
```

Repository layout: `configs/`, `data/`, `src/`, `scripts/`, `docs/`, `tests/`.
